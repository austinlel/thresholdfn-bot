ThresholdFN Matchmaking Bot

Discord bot for Fortnite competitive matchmaking.

Features

  • ELO-based matchmaking
  • 1v1, 2v2, 3v3, 4v4 modes
  • Statistics tracking
  • Leaderboards

Privacy Policy

Information We Collect:

  • Discord User ID and Username
  • Epic Games Account ID and Display Name
  • Match statistics (wins, losses, ELO)

Information We Don't Collect:

  • Passwords
  • Payment information
  • Personal identification data

Data Usage:
Your data is used solely for matchmaking, ranking, and statistics.

Your Rights:
You can delete your data anytime using the /unlink command.

Contact: Discord Server
